Guide d'Installation - Optimisation Réseau Ebolowa

Installation sous Windows

1. Installation de Python 3.11

  • Téléchargez Python 3.11 depuis python.org
  • Sélectionnez "Windows installer (64-bit)"
  • Exécutez l'installateur
  • ⚠️ Important : Cochez "Add Python 3.11 to PATH"

2. Création de l'environnement virtuel

# Ouvrez cmd et créez un dossier pour votre projet
mkdir ebolowa_optim
cd ebolowa_optim

# Créez l'environnement virtuel
python -m venv venv

# Activez l'environnement virtuel
venv\Scripts\activate

3. Installation des packages

# Mise à jour de pip
python -m pip install --upgrade pip

# Installation des dépendances
pip install numpy pandas matplotlib seaborn deap wntr xlsxwriter

Installation sous Linux

1. Installation de Python 3.11

# Ubuntu/Debian
sudo apt update
sudo apt install python3.11 python3.11-venv

# Fedora
sudo dnf install python3.11 python3.11-devel

2. Création de l'environnement virtuel

# Créez un dossier pour votre projet
mkdir ebolowa_optim
cd ebolowa_optim

# Créez l'environnement virtuel
python3.11 -m venv venv

# Activez l'environnement virtuel
source venv/bin/activate

3. Installation des packages

# Mise à jour de pip
pip install --upgrade pip

# Installation des dépendances système
# Ubuntu/Debian
sudo apt install build-essential python3-dev python3-tk

# Fedora
sudo dnf groupinstall "Development Tools"
sudo dnf install python3-tkinter

# Installation des dépendances Python
pip install numpy pandas matplotlib seaborn deap wntr xlsxwriter

Configuration du Projet

# Créez les dossiers requis
mkdir visualisation rapports

# Créez les fichiers Python
# Copiez le contenu fourni dans :
# - code1.py
# - gui.py
# - optimisation.py
# - optimisation_cli.py

Exécution

# Pour l'interface graphique
python gui.py

# Pour la version CLI
python optimisation_cli.py

Vérification de l'Installation

python -c "import numpy; import pandas; import matplotlib; import seaborn; import deap; import wntr; print('Tout est bien installé!')"
Notes importantes :
  • Assurez-vous que le fichier 'ebolowa_reseau.inp' est présent dans le dossier du projet
  • Les dossiers 'visualisation' et 'rapports' doivent avoir les permissions d'écriture appropriées
  • Si vous rencontrez des erreurs, vérifiez les messages d'erreur pour les versions des packages
  • Sur Windows, redémarrez votre terminal après l'installation de Python
Résolution des problèmes courants :
  • Si Python n'est pas reconnu : vérifiez que Python est bien dans le PATH
  • Si un package ne s'installe pas : essayez d'installer ses dépendances système d'abord
  • Pour les erreurs tkinter : assurez-vous que python3-tk est installé
